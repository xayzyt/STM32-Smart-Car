# STM32 Smart Car

基于 STM32 的智能小车项目，包含硬件资料、原理图/PCB、软件源码和学习笔记。项目面向嵌入式入门与综合实践，覆盖传感器读取、电机控制、蓝牙控制和多种运行模式。

## 功能特性

- 超声波跟随
- 超声波避障
- 红外循迹
- 蓝牙遥控
- 电机驱动与基础运动控制
- 原理图、PCB 和配套资料整理

## 技术栈

- STM32F10x
- C 语言
- Keil / STM32 标准外设库
- 超声波模块
- 红外循迹模块
- 蓝牙通信模块
- 电机驱动模块

## 仓库结构

```text
01程序资料/    STM32 工程源码与外设库
02笔记资料/    项目笔记和调试记录
```

## 项目价值

这个项目展示了一个完整嵌入式小车从硬件资料、驱动代码到功能调试的实践过程，适合作为 STM32、传感器和电机控制方向的入门项目参考。
